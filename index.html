AES-256-GCM

Advanced Encryption Standard in Galois/Counter Mode. Symmetric cipher — same key encrypts and decrypts. Each operation generates a random 128-bit salt + 96-bit nonce. The password is stretched via PBKDF2-SHA256 (480,000 iterations) to produce a 256-bit key. GCM mode provides authenticated encryption — tampering is detected.

Key size    → 256 bits
Mode       → GCM (authenticated)
KDF        → PBKDF2-SHA256
Salt        → 16 bytes (random)
Nonce      → 12 bytes (random)
RSA-2048 / OAEP

Rivest–Shamir–Adleman asymmetric encryption. Two mathematically linked keys — public key encrypts, private key decrypts. Uses OAEP padding with SHA-256 for security. Suitable for encrypting small payloads or symmetric keys (hybrid encryption pattern).

Key sizes   → 2048 / 4096 bits
Padding    → OAEP-SHA256
Exponent   → 65537
Format     → PEM (PKCS#1)
CLI USAGE
# Install dependency
pip install cryptography

# AES encrypt
python vaultcipher_cli.py aes-encrypt --text "secret" --password "mypass"

# Generate RSA keys
python vaultcipher_cli.py rsa-keygen --bits 2048 --output ./keys

# RSA encrypt
python vaultcipher_cli.py rsa-encrypt --text "hello" --pubkey ./keys/public_key.pem

# RSA decrypt
python vaultcipher_cli.py rsa-decrypt --payload "BASE64..." --privkey ./keys/private_key.pem
